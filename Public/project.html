My Project

Real Time Cost Minimization of Fog Computing in
Mobile-Base Station Using Networked Disaster Areas

Institution: LRG Government Arts and Science Women's College, Tirupur

Duration: 2023 - 2024

Submitted Date: 21 March

Project Description

Traditional techniques for image retrieval are not supported for the ever-expansive image database.
These downsides can be removed by utilizing contents of the image for image retrieval, called Cross Batch Redundancy Detection (CBRD).
BEES works with CBRD focused on visual features like shape, color, and texture.
The Density-Bandwidth Energy Efficient Sharing (BEES) is a locally feature detector and descriptor used in most vision programming.
Focus on texture, color, shape, size, and string-based image matching with better accuracy.

Technologies Used

  • Front end: Net Beans IDE 8.3/JDK 1.8
  • Coding Language: JAVA
